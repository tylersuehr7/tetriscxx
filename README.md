# Tetris

A re-implementation of Tetris in the C++ 20 programming language.

## Dependencies
- C++ 20
- `raylib`
- `cmake`
- `make`
- `git`

## How to build this project
1. Create build directory and initialize cmake:
    - `$ mkdir -p build && cd build && cmake ..`
2. Build the project:
    - `$ make`
3. Run the executable:
    - `$ ./tetris`
